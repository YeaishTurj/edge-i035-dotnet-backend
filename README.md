# EDGE Course — Backend (I035)

Short backend repository for EDGE Batch I035 deliverables (SSIS + SSRS).

- `ProjectI035/Integration Services Project1`: SSIS packages, project files, and `.ispac`
- `ProjectI035/Report Project1`: SSRS report project and report definitions

Open with Visual Studio + SSDT to build/deploy. Keep credentials and connection strings in deployment parameters, not in Git.
